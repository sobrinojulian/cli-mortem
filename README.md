# mortem-cli

> Morbid stats


## Install

```
$ npm install --global mortem-cli
```


## Usage

```
$ mortem --help

  Usage
    $ mortem <command>

  Commands
    sbd   Set Birth Date. FORMAT: YYYY/MM/DD
    gbd   Get Birth Date
    ndl   Number of Days Lived
    pro   Progress Percentage
    etr   Estimated Time Remaining
    eyd   Estimated Date of Death

  Examples
    $ mortem sbd 1992/07/07
    $ mortem gbd
    1992/07/07
    $ mortem ndl
    9363
    $ mortem pro
    36.62%
    $ mortem etr
    16204
    $ mortem eyd
    2062
```
